# scripts

Build, test, and development scripts. Key scripts: setup.sh (init submodules + build GhosttyKit), reload.sh --tag (build + launch debug app), reloadp.sh (release app), bump-version.sh (version + build number), build-sign-upload.sh (release signing). Test runners: run-tests-v2.sh, run-e2e.sh, test-unit.sh. See root CLAUDE.md for usage.
