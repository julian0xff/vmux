# Services

System integration services extracted from AppDelegate.

- `InputRouting.swift` — keyboard routing helpers (omnibar navigation, command palette, shortcut suppression)
- `MenuBarService.swift` — menu bar badge/icon state
- `CLIInstallService.swift` — vmux CLI installation management
- `VSCodeIntegration.swift` — VS Code serve-web integration
- `TypingPerformance.swift` — keystroke latency monitoring
- `WorkspaceShortcuts.swift` — Cmd+digit shortcut mapping
