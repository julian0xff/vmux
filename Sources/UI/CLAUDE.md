# UI

UI behaviors extracted from ContentView.

- `CommandPaletteSearch.swift` — fuzzy search engine and indexer for the command palette
- `SidebarDragDrop.swift` — drag lifecycle, drop planning, and auto-scroll
- `SidebarAppearance.swift` — material/blend/tint options
- `WindowEffects.swift` — NSGlassEffectView and SidebarState
