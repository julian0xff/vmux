# Sources

Main app target source code. ContentView.swift is the root composition view (sidebar + workspace area). Organized into subdirectories: App/ (lifecycle), Window/ (portals, chrome), Workspace/ (model layer), Terminal/ (Ghostty engine, socket server), Panels/ (terminal/browser/markdown), Notifications/ (bell/badge system), Services/ (system integration), UI/ (extracted behaviors), Find/ (search overlays), Update/ (titlebar accessory), Utilities/ (small helpers).
