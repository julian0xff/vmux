# VmuxCore

Foundation package. Shared protocols, types, and settings used by all other packages.

- `Panel.swift` — Panel protocol and focus types
- `TerminalEngineProtocol.swift` — abstract Ghostty interface
- `KeyboardShortcutSettings.swift` — shortcut action definitions
- `GhosttyConfig.swift` — terminal color/font/theme parsing
- `SidebarFolderModel.swift` + `SidebarFolderStore.swift` — sidebar tree model
- `GhosttyNotifications.swift` — notification names
- `Backport.swift` — SwiftUI compatibility helpers
